## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/woabu0/devign.git
    ```
2. Navigate to the project directory:
    ```bash
    cd devign
    ```
3. Install dependencies:
    ```bash
    npm add vite@latest
    ```

## Usage
1. Start the application:
    ```bash
    npm run dev
    ```
2. Open your browser and navigate to:
    ```
    http://localhost:5173
    ```
